# SQL Mastery Coach Agent (Project-Based)

A project-enforced SQL learning agent designed for college students (India-first clarity).  
Includes self-assessment, timed practice, mandatory portfolio milestones, and interview role-play.

## What’s inside this repo
- `prompts/` — the tutor prompt file students upload into ChatGPT
- `docs/` — student onboarding + progress log instructions
- `project-creation-helper/` — *open only when Project 1 begins* (free tools, Kaggle, GitHub)
- `pdf/` — downloadable PDFs (program pack, instructions, etc.)

## Quick start (for students)
1. Download `prompts/sql_mastery_coach_prompt.txt`
2. Upload it into ChatGPT in a new chat
3. Type: **Start the SQL Mastery Coach.**
4. Maintain `sql_progress_log.md` as instructed in `docs/`
